模板就是一个包含响应文本的文件，其中包含用占位变量表示的动态部分。python使用的Jinja2模板
有几个注意点
```
  {% ... %} 语句（Statements）
  {{ ... }} 打印模板输出的表达式（Expressions）
  {# ... #} 注释
```
